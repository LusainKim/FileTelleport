연결된 네트워크에서 파일을 손쉽게 업로드 및 다운로드를 할 수 있는 프로그램입니다.

가능한 것 : 파일 전송 및 업로드
불가능한 것 : 폴더 전송(압축 필수), 서버 프로그램이 파일을 저장하는 폴더 필요

사용법 : 
  
  클라이언트는 좌측의 리스트에서 다운로드를 원하는 파일을 선택하면 저장 폴더를 지정하는 창이 뜹니다.

  서버는 서버 프로그램이 위치하는 폴더에 Download라는 폴더가 있어야 합니다.
  배포를 원하는 파일을 Download 폴더에 넣으면 됩니다.

  클라이언트 프로그램에 업로드를 원하는 파일을 드래그하면 서버로 업로드 할 수 있습니다.
  단, 업로드 위치는 서버 프로그램이 있는 폴더의 Download 폴더입니다.

  ※ 업로드는 파일의 중복 검사를 하지 않습니다.

주의 : 
  프로그램은 최적화가 되어있지 않습니다.
  다운로드/업로드 중 프로그램을 종료하지 마세요. 정상적인 동작을 보장하지 못합니다.
